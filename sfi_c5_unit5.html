SFI C级 · 第五单元 · 语法核心
阅读与文本分析
掌握关系从句（som/vars/vilket）、-s被动语态，以及阅读理解常用句式，能够分析和讨论文章内容。
① 关系代词 som ② 所有格 vars ③ 句子指代 vilket ④ 被动语态 -s ⑤ 阅读句式 ⏱ 约50分钟
① 关系从句基础：som
som 是最常用的关系代词，可以替代主语（人/物）或宾语（人/物）。
基本结构：主句名词 + som + 从句（缺少主语或宾语）
Mannen som arbetar här heter Anders. → 在这里工作的那个男人叫安德斯。
Artikeln som vi läste igår var intressant. → 我们昨天读的那篇文章很有意思。
Texten som handlar om hälsa är viktig. → 关于健康的那篇文章很重要。
som 替代主语（主语关系从句）
Eleven som svarade rätt fick ett pris.
答对了的那个学生得了奖。
Källan som används är pålitlig.
被使用的那个来源是可信的。
som 替代宾语（宾语关系从句）
Boken som jag rekommenderar är bra.
我推荐的那本书很好。
Argumentet som hon nämnde var starkt.
她提到的那个论点很有力。
💡 记忆技巧：som 后面的从句里缺少一个名词成分（主语或宾语），这个缺失的成分就是前面的名词。
✅ Texten som vi läste（缺宾语：vi läste texten → texten som vi läste）
✅ Texten som handlar om hälsa（缺主语：texten handlar → texten som handlar）
② 所有格关系代词：vars
vars 表示"谁的/它的"，用于说明前面名词的所属关系，相当于英语 whose。
基本结构：主句名词 + vars + 属于该名词的事物
Eleven vars uppsats var bäst fick ett pris. → 作文最好的那个学生得了奖。
Författaren vars böcker jag läser är känd. → 我读他书的那位作家很有名。
Huset vars fönster är röda är gammalt. → 窗户是红色的那栋房子很旧。
⚠️ vars 与 som 的区别：
Eleven som svarade rätt... → 关系到学生的动作（答对了的学生）
Eleven vars svar var rätt... → 关系到学生拥有的东西（答案是对的那个学生）
简单记忆：vars 后面紧跟"属于前面名词的东西"，相当于 hans/hennes/dess。
③ 句子指代关系代词：vilket
vilket 用于指代整个前面的句子或情况，不是指某个具体名词，常放在逗号后。
基本结构：完整句子，+ vilket + 对整句的说明
Hon klarade provet, vilket var fantastiskt. → 她通过了考试，这真太棒了。
Han kom för sent, vilket gjorde chefen arg. → 他来晚了，这让老板很生气。
De bestämde sig för att flytta, vilket alla förstod. → 他们决定搬家，这大家都理解。
✅ vilket（指代整句）
Hon läste texten, vilket tog lång tid.
她读了文章，这花了很长时间。
Det regnade hela dagen, vilket var tråkigt.
整天都在下雨，这很无聊。
❌ 不能用 som 指代整句
✗ Hon klarade provet, som var fantastiskt.
（som 只能指代具体名词，不能指代整件事）
✅ Hon klarade provet, vilket var fantastiskt.
🎯 SFI考试提示：
som → 前有名词作先行词（人/物）
vars → 前有名词，表达所属（谁的……）
vilket → 前有整个句子，通常紧接逗号
④ 被动语态 -s 形式
-s被动语态是瑞典语特有的简洁被动结构，在书面语和正式场合频繁出现。
构成规则：现在时去掉 -r，加 -s
läser → läses（被阅读）· skriver → skrivs（被写）
arbetar → arbetas（被工作于/被执行）· använder → används（被使用）
finansierar → finansieras（被资助）
原形主动（现在时）-s被动（现在时）中文（被动义）
att läsaläserläses被阅读
att skrivaskriverskrivs被书写
att användaanvänderanvänds被使用
att finansierafinansierarfinansieras被资助
att remitteraremitterarremitteras被转介
att diskuteradiskuterardiskuteras被讨论
过去时 -s 被动
läste → lästes（被阅读了）
skickade → skickades（被发送了）
diskuterade → diskuterades（被讨论了）
常见例句
Texten läses av alla elever.
文章被所有学生阅读。
Rapporten skickades igår.
报告昨天被发送了。
📝 注意：skriva→skrivs（-er变-s，不是-ers）；有些动词去掉-er直接加-s：
arbetar→arbetas，läser→läses，skriver→skrivs，använder→används。
规律：去掉现在时词尾 -r，加 -s。若词尾为 -er，则变成 -es（läser→läses）。
⑤ 阅读理解常用句式
这些句式在SFI阅读测试和写作中频繁使用，掌握它们有助于快速理解文章结构。
瑞典语句式中文含义用法说明
Texten handlar om...文章讲的是……概述文章主题
Författaren påstår att...作者声称……引用作者观点
Enligt texten...根据文章……引用文章内容
Det framgår av texten att...文章表明……说明文章表达的信息
Sammanfattningsvis...总的来说……总结段落或文章
Å ena sidan... å andra sidan...一方面……另一方面……对比论述
Texten avslutas med att...文章以……结尾描述文章结尾
🎯 SFI写作模板：
Texten handlar om [主题]. Författaren påstår att [主要观点]. Enligt texten [具体内容]. Sammanfattningsvis [总结].

好！你认为文章中哪个论点最有力？
🧑‍🎓

我认为关于平等医疗的论点最有力。文章声称所有人都有权获得同等医疗，这感觉很公平。

Utmärkt analys! Kan du nämna ett nyckelord som används genomgående i texten?
分析得很好！你能说出一个在文章中贯穿始终使用的关键词吗？
🧑‍🎓
"Jämlikhet!" Det är ett ord som förekommer i varje stycke. Sammanfattningsvis beskriver texten ett system vars mål är att ge alla lika vård.
"平等！"这是一个出现在每个段落的词。总的来说，文章描述的是一个目标是给所有人平等医疗的系统。

📄 阅读文段（回答第8–10题时参考）

Vården i Sverige

Hälso- och sjukvården i Sverige finansieras av skatterna och är tillgänglig för alla invånare. Det svenska vårdsystemet anses vara ett av de bästa i världen. Alla medborgare har rätt till jämlik vård, vilket innebär att ingen nekas vård på grund av ekonomiska skäl.

Primärvården, som utgör grunden i systemet, erbjuder grundläggande medicinska tjänster. Patienter vars tillstånd är allvarliga remitteras till specialister. Systemet finansieras genom skatteintäkter, vilket gör att kostnaden för enskilda patienter hålls låg.

SFI C级 · 第一单元 · 语法核心
过去时 Preteritum
掌握瑞典语四大动词变化组的过去时形式，以及常用不规则动词，能够描述过去发生的事情。
① 第一组 -ade ② 第二组 -te/-de ③ 第三组 -dde ④ 不规则动词 ⏱ 约50分钟
📌 什么是过去时（Preteritum）？
过去时用于描述已经完成的动作或过去的状态。
基本结构：主语 + 动词过去时 + 其他成分
Jag arbetade igår. → 我昨天工作了。
Hon åt frukost klockan åtta. → 她八点吃了早饭。
Vi bodde i Malmö förra året. → 我们去年住在马尔默。
⏰ 常用时间词（配合过去时使用）：
igår（昨天）· förra veckan（上周）· förra månaden（上个月）· förra året（去年）
för två dagar sedan（两天前）· då（那时）· nyss（刚才）· tidigare（之前）
① 第一组动词：过去时加 -ade
最常见的一组，原形以-a结尾，现在时-ar，过去时加-ade。
规律：动词原形（去掉-a）+ -ade
原形（Infinitiv）现在时（Presens）过去时（Preteritum）中文
att arbetaarbetararbetade工作
att stannastannarstannade停留/停下
att tittatittartittade看/注视
att lyssnalyssnarlyssnade听
att pratapratarpratade说话
att hälsahälsarhälsade问候/致意
💡 记忆技巧：第一组词量最大，约占所有动词60%。看到-ar结尾的现在时，过去时多半加-ade。
例外：第二组的一些动词也用-er但过去时不是-ade，注意辨别。
② 第二组动词：过去时加 -te 或 -de
现在时-er结尾，过去时根据词干末尾辅音选择-te或-de。
2a组：词干以清辅音结尾 → -te
läsa → läste（读）
köpa → köpte（买）
möta → mötte（见面）
stänga → stängde（关）
välja → valde（选择）
2b组：词干以浊辅音/元音结尾 → -de
ringa → ringde（打电话）
leva → levde（生活）
läsa → läste（注意：具体看词干）
stänga → stängde（关闭）
glömma → glömde（忘记）
SFI简化记忆法：
清辅音（k, p, t, s, x）结尾 → 加-te：läste, köpte, mötte
浊辅音（ng, m, v）结尾 → 加-de：ringde, levde, glömde
遇到不确定的，查词典或背诵为主！
③ 第三组动词：过去时加 -dde
原形为单音节，以元音结尾，现在时-r，过去时加-dde。
原形现在时过去时中文
att boborbodde住/居住
att trotrortrodde相信/认为
att sysyrsydde缝纫
att nånårnådde到达/联系到
📝 第三组特征：原形极短（bo, tro, sy, nå），现在时直接加-r，过去时加-dde（双写d）。数量较少，容易掌握。
④ 不规则动词（强变化）：元音替换
第四组（强变化动词）通过元音交替变化，没有规律性词尾，需单独记忆。
原形现在时过去时中文
att varaärvar是/在
att haharhade有
att bliblirblev变成/成为
att gågårgick走/去
att kommakommerkom来
att tatartog拿/取
att sesersåg看见
att ätaäteråt吃
att drickadrickerdrack喝
att skrivaskriverskrev写
att sägasägersa/sade说
att vetavetvisste知道
SFI必背不规则动词：var（是/在）、hade（有）、blev（变成）、gick（走）、kom（来）、tog（拿）、såg（看见）、åt（吃）。这8个是口语和写作考试最高频的过去时形式！
🔄 过去时与完成时的区别
过去时 Preteritum
表示过去某时发生，已结束
Jag åt frukost igår.
我昨天吃了早饭。（完成了）
Hon bodde i Stockholm.
她（过去）住在斯德哥尔摩。（现在不住了）
完成时 Perfekt（预告：C级第2单元）
表示与现在有关联的过去
Jag har ätit frukost.
我吃过早饭了。（已经吃了）
Hon har bott i Stockholm.
她住过斯德哥尔摩。（经历）
🎯 SFI考试提示：看到igår、förra veckan等明确过去时间词，用过去时（preteritum）。
看到redan（已经）、någon gång（曾经）等，通常用完成时（perfekt）。

💬 对话一：周末发生了什么？
朋友之间谈论周末经历。金色=过去时动词，蓝色=时间词。点击任意单词可听发音。
🧑
Hej! Vad gjorde du i helgen?
嗨！你周末做了什么？
👩
Jag åkte till Göteborg med familjen. Vi bodde på hotell och besökte Liseberg.
我和家人去了哥德堡。我们住在酒店，参观了利斯堡游乐园。
🧑
Vad roligt! Åt ni något speciellt?
真好玩！你们吃了什么特别的东西吗？
👩
Ja! Vi åt fisk och räkor vid hamnen. Det var jättegott! Barnen drack saft och vi vuxna drack vin.
是的！我们在港口吃了鱼和虾。超级好吃！孩子们喝了果汁，我们大人喝了葡萄酒。
🧑
Träffade ni några gamla vänner?
你们见到了老朋友吗？
👩
Ja, vi träffade min gamla klasskompis. Vi pratade i flera timmar och kom ihåg massa saker från skolan.
是的，我们见到了我的老同学。我们聊了好几个小时，想起了很多学校里的事情。
⏰ 时间词：i helgen（周末）
⚡ 过去时：gjorde（做了）、åkte（去了）、bodde（住了）、besökte（参观了）、åt（吃了）、var（是/很）、drack（喝了）、träffade（见到了）、pratade（聊了）、kom ihåg（想起了）
💬 对话二：昨天在公司
同事之间谈论昨天的工作情况。SFI C级常见职场情境。
👨‍💼
Du var inte på mötet igår. Vad hände?
你昨天没来开会。发生什么了？
👩‍💼
Förlåt, jag glömde det. Jag hade ont i magen och stannade hemma.
对不起，我忘了。我肚子不舒服，就待在家里了。
👨‍💼
Okej. Vi diskuterade det nya projektet. Chefen presenterade planen och alla tyckte att det lät bra.
好吧。我们讨论了新项目。老板介绍了计划，大家都觉得听起来不错。
👩‍💼
Bestämde ni något?
你们决定了什么吗？
👨‍💼
Ja, vi bestämde att starta projektet nästa månad. Jag skickade ett mejl med all information. Fick du det?
是的，我们决定下个月启动项目。我发了一封包含所有信息的邮件。你收到了吗？
👩‍💼
Ja, tack! Jag läste det just. Det verkade intressant!
是的，谢谢！我刚读了。看起来很有趣！
⚡ 过去时：var（是）、hände（发生了）、glömde（忘了）、hade（有/感到）、stannade（待了）、diskuterade（讨论了）、presenterade（介绍了）、tyckte（觉得）、bestämde（决定了）、skickade（发了）、fick（收到了）、läste（读了）、verkade（看起来）

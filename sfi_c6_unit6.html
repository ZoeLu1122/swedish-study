掌握段落连接词（Konnektorer）、正式书写句式，以及三段式写作结构，能够写出逻辑清晰、表达规范的瑞典语文章。
① 段落连接词 ② 正式书写句式 ③ 三段式写作结构 ⏱ 约50分钟
① 段落连接词 Konnektorer
连接词是写作的"粘合剂"，用于衔接句子与段落，让文章逻辑清晰、表达流畅。
功能连接词中文含义示例
递进 dessutom / även / också 此外 / 也 / 也 Dessutom är det billigare.
转折 men / däremot / å andra sidan 但是 / 相反 / 另一方面 Däremot är det tidskrävande.
因果 därför / eftersom / på grund av 因此 / 因为 / 由于 Eftersom det regnar, stannar vi inne.
举例 till exempel / bland annat 例如 / 其中包括 Till exempel kan man cykla.
总结 sammanfattningsvis / alltså / till slut 总的来说 / 所以 / 最后 Sammanfattningsvis anser jag att…
顺序 först / sedan / därefter / slutligen 首先 / 然后 / 之后 / 最后 Först presenteras problemet.
💡 书写技巧：不要在同一段落中重复使用同一个连接词。交替使用 dessutom / även / också 可以使文章更多样。
⚠️ 常见错误：口语中常用 "och sen"（然后），书面语应换成 dessutom（此外）或 därefter（之后）。
② 正式书写句式
SFI C级写作要求使用正式语气。以下是常见的非正式→正式表达对照。
非正式（口语）正式（书面语）说明
Jag tycker… Enligt min åsikt… / Jag anser att… 表达个人观点
Det är bra. Det är fördelaktigt. 用更正式的形容词
Det är dåligt. Det är negativt / skadligt. 替换口语形容词
Jag vill säga att… Jag vill påpeka att… 正式指出某事
Och sen… Dessutom… / Vidare… 正式递进连接词
Jag tror… Jag hävdar att… / Det kan hävdas att… 强化论述语气
正式写作的四条原则：
1. 用 Enligt min åsikt 或 Jag anser att 替代 "Jag tycker"
2. 用 dessutom / vidare / däremot 替代 "och sen / men"
3. 用 fördelaktigt / negativt / skadligt 替代 "bra / dåligt"
4. 避免缩写：写 det är，不写 det's；写 till exempel，不写 t.ex.（在正文中）
③ 写作结构：三段式 Trepunktsmodellen
瑞典语写作（作文、信件、报告）通常遵循三段式结构。
① Inledning（引言）
介绍主题
说明写作目的

句式模板：
Syftet med detta brev är att…
I denna text vill jag diskutera…
② Innehåll（主体）
展开论点
举例说明
使用连接词串联

句式模板：
Till exempel…
Dessutom…
③ Avslutning（结尾）
总结观点
提出建议（可选）

句式模板：
Sammanfattningsvis anser jag att…
Därför föreslår jag att…
完整段落示例（关于城市绿化）：
① Syftet med detta brev är att diskutera miljön i vår stad.
② Många invånare saknar tillgång till grönområden. Dessutom mår människor bättre när de har tillgång till natur. Till exempel kan man plantera träd längs gatorna.
③ Sammanfattningsvis anser jag att mer grönska behövs i staden.
🎯 SFI考试写作提示：确保每段都有明确的功能，引言不超过2句，主体用连接词连接至少2个论点，结尾用 sammanfattningsvis 开头总结。

💬 对话一：讨论写作任务
同学之间讨论如何完成一篇写给市政局的信件。玫红色=连接词，蓝色=正式书写用语。
🧑
Vad ska vi skriva om den här veckan?
我们这周要写什么？
👩
Vi ska skriva ett brev till kommunen om parken i vårt område. Det är viktigt att inleda med att förklara syftet.
我们要给市政局写一封关于我们社区公园的信。以说明目的来开头很重要。
🧑
Hur ska jag börja? Kan jag skriva "Jag tycker att parken är dålig"?
我怎么开头？我可以写"我觉得公园很差"吗？
👩
Nej, det är för informellt. Skriv istället: "Syftet med detta brev är att påpeka att parken behöver upprustning." Dessutom bör du använda enligt min åsikt istället för "Jag tycker".
不，那太口语化了。可以换成："本信的目的是指出公园需要翻新。"此外，你应该用"根据我的看法"来替代"我觉得"。
🧑
Och hur avslutar man brevet?
那信怎么结尾呢？
👩
Sammanfatta dina synpunkter och föreslå en lösning. Använd "Sammanfattningsvis" eller "Därför anser jag att…"
总结你的观点并提出一个解决方案。用"总的来说"或"因此我认为……"
🔴 连接词：Dessutom（此外）、Sammanfattningsvis（总的来说）、Därför（因此）
🔵 正式用语：syftet（目的）、inleda（引入）、påpeka（指出）、synpunkter（观点）、föreslå（建议）、according to min åsikt（根据我的看法）、avslutar（结束）
💬 对话二：互评作文
两位同学互相批改作文，讨论如何改进段落结构和连接词使用。
🧑‍💼
Jag har läst din text. Den är bra, men den saknar ett tydligt avslut.
我读了你的文章。写得不错，但缺少一个清晰的结尾。
👩‍💼
Ja, jag vet. Jag hade svårt att sammanfatta mina synpunkter på ett bra sätt.
是的，我知道。我很难把我的观点用好的方式总结起来。
🧑‍💼
Prova att börja avslutningsstycket med "Sammanfattningsvis anser jag att…" Det ger texten en tydlig slutsats.
试着用"总的来说我认为……"来开始结尾段落。这会给文章一个清晰的结论。
👩‍💼
Bra idé! Dessutom kanske jag borde ge ett exempel i det andra stycket? Jag hävdar att parken behövs, men jag ger inga bevis.
好主意！此外，我是不是应该在第二段给一个例子？我主张公园是必要的，但没有给出任何证据。
🧑‍💼
Precis. Till exempel kan du skriva om din egen erfarenhet. Det är ett bra sätt att motivera ditt argument.
正是。比如你可以写写你自己的经历。这是证明你论点的好方法。
👩‍💼
Tack! Det gör texten mer övertygande. Jag ska också byta ut "Det är dåligt" mot "Det är negativt" – det låter mer formellt.
谢谢！这样文章会更有说服力。我还要把"这很差"换成"这是负面的"——听起来更正式。
🔴 连接词：men（但是）、Dessutom（此外）、Sammanfattningsvis（总的来说）、Till exempel（比如）
🔵 正式用语：avslut（结尾）、synpunkter（观点）、anser jag att（我认为）、slutsats（结论）、stycke（段落）、hävdar（主张）、motivera（论证）、exempel（例子）

# B3 标准模板与复制清单

本文档把 B3 当前稳定结构整理成模板。以后新增 B4、B5、C、D 单元时，先参考这里，不要凭记忆复制。

## 1. B3 当前标准文件

B3 由这些核心文件组成：

```text
sfi_b3_unit3.html
sfi_practice_B_unit3.html
b3_unit3_bank.js
audio/sv/b3/unit3/audio-manifest.json
audio/sv/b3/unit3/content-manifest.json
01 音频整理/瑞典语学习-音频整理.xlsx
01 音频整理/b3_unit3_mapping_overrides.json
```

公共文件：

```text
common.css
common.js
practice-common.js
```

工具文件：

```text
02 tools/check_unit.py
02 tools/build_unit_audio_manifests.py
02 tools/export_used_audio_mapping.py
```

## 2. 新单元复制时，哪些文件要复制

如果以后从 B3 复制一个新单元，例如 B4，通常需要复制：

```text
sfi_b3_unit3.html -> sfi_b4_unit4.html
sfi_practice_B_unit3.html -> sfi_practice_B_unit4.html
b3_unit3_bank.js -> b4_unit4_bank.js
```

还需要新建音频文件夹：

```text
audio/sv/b4/unit4/
  vocab/
  dialogue/
  question/
  audio-manifest.json
  content-manifest.json
```

不要复制公共文件：

```text
common.css
common.js
practice-common.js
```

这些是全平台共享文件，除非明确要改平台能力，否则不要为了某个新单元单独复制。

## 3. 复制后必须修改的地方

以 B3 复制为 B4 为例，必须修改：

```text
b3 -> b4
B3 -> B4
unit3 -> unit4
Unit3 -> Unit4
第三单元 -> 第四单元
UNIT_BANK_B3 -> UNIT_BANK_B4
b3_unit3_bank.js -> b4_unit4_bank.js
sfi_b3_unit3.html -> sfi_b4_unit4.html
sfi_practice_B_unit3.html -> sfi_practice_B_unit4.html
audio/sv/b3/unit3 -> audio/sv/b4/unit4
```

练习页里特别要改：

```text
script.src = 'b3_unit3_bank.js'
window.UNIT_BANK_B3
Bank object not found: UNIT_BANK_B3
Failed to load bank file: b3_unit3_bank.js
```

题库文件里特别要改：

```text
window.UNIT_BANK_B3
unit: 'b3'
所有题目 id 前缀：b3_...
```

学习页里特别要改：

```text
<title>
导航栏单元名
页面标题
UNIT_KEY='sfi_b3_unit3'
对话 contentId，例如 b3_dialog_line_...
词汇数据里的 audioRef
```

首页里特别要改：

```text
进入学习链接
开始练习链接
单元标题
完成状态
```

## 4. 复制后不能乱改的地方

不要随意改：

```text
common.js
common.css
practice-common.js
audio 文件夹名
audio-manifest.json 的结构
content-manifest.json 的结构
题库字段名：id / kind / source / lemma / prompt / answer / template
```

这些地方一旦改错，可能影响多个单元。

## 5. B3 题库结构

B3 题库文件是：

```text
b3_unit3_bank.js
```

开头必须类似：

```js
window.UNIT_BANK_B3 = {
  unit: 'b3',
  unitTitle: '名词复数变化',
  basicPool: [],
  advancedPool: []
};
```

`basicPool` 用于基础模式。

常见题型：

```text
vocab_spelling
单词拼写题。

phrase_fill
固定表达填空题。

dialog_fill
对话句填空题。
```

`advancedPool` 用于进阶模式。

B3 当前重点题型：

```text
noun_forms
名词变形。

pattern
数量结构或固定结构专项。
```

## 6. B3 名词变形题规则

B3 名词变形题示例：

```js
{
  id: 'b3_noun_vecka_group',
  kind: 'morphology',
  source: 'noun_forms',
  labelScheme: 'noun_forms',
  lemma: 'vecka',
  prompt: { zh: '周' },
  template: '{{en}} | {{vecka}} | {{veckan}} | {{veckor}} | {{veckorna}}'
}
```

页面显示：

```text
en / ett + 单数不定式
单数定式
复数不定式
复数定式
```

音频规则：

```text
界面可以保留多个空。
但 en/ett + 名词原形 优先使用一条完整自然音频。
后面的词形仍然要能独立播放。
```

例如：

```text
en stol
stolen
stolar
stolarna
```

推荐音频：

```text
en stol 用一条完整 mp3
stolen 用一条 mp3
stolar 用一条 mp3
stolarna 用一条 mp3
```

不推荐：

```text
en + stol 两个音频拼接
```

## 7. 单词页发音规则

B3 学习页里的单词数据直接写在：

```text
sfi_b3_unit3.html
```

每个单词通常有 4 个音频引用：

```text
headAudioRef
defAudioRef
plAudioRef
plDefAudioRef
```

例如：

```text
en månad
månaden
månader
månaderna
```

这些引用必须能在：

```text
audio/sv/b3/unit3/audio-manifest.json
```

中找到对应 `audioId`。

如果点击单词没声音，优先检查：

```text
页面里的 audioRef 是否存在于 audio-manifest.json
audio-manifest.json 的 filePath 是否存在 mp3
```

## 8. manifest 规则

网页实际播放音频依赖：

```text
audio-manifest.json
content-manifest.json
```

Excel 是台账，manifest 是网页运行清单。

不能只改 Excel，不更新 manifest。
也不能只放 mp3，不更新 Excel 和 manifest。

## 9. B3 当前允许的特殊情况

B3 允许跨单元复用音频，例如：

```text
B3 使用 A2 或 B1 已有音频
```

检查工具会提示 cross-unit warning，这不一定是错误。

只要满足：

```text
源 mp3 文件存在
audio-manifest.json 能找到
content-manifest.json 引用正确
本地测试能播放
```

就可以接受。

## 10. 新单元完成前的检查清单

新单元上线前检查：

```text
学习页能打开
练习页能打开
题库文件能加载
单词页点击发音正常
对话页音频正常
基础练习音频正常
进阶练习音频正常
测验页显示清爽
audio-manifest.json 存在
content-manifest.json 存在
Excel 已记录音频
检查工具 ERROR=0
GitHub 上传后线上能打开
```

B3 检查命令：

```bash
python3 "Swedish study/02 tools/check_unit.py" --unit b3 --skip-index
```


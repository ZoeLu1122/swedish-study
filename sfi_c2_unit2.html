📌 什么是情态动词（Modala verb）？
情态动词表达说话者的态度——能力、意愿、义务、许可、建议。它们后面接动词原形，且不带 att。
基本结构：主语 + 情态动词 + 动词原形（不带att）
Jag kan simma. → 我会游泳。
Du måste studera. → 你必须学习。
Vi ska resa till Sverige. → 我们将去瑞典旅行。
⚠️ 核心规则：情态动词后接原形（infinitiv），且绝对不加 att！
✅ Jag vill äta.  ❌ Jag vill att äta.  ❌ Jag vill äter.

💡 特别注意：
· måste 的现在时和过去时形式完全相同，都是 måste。
· ska 在口语中常省略原形 skola，直接使用 ska / skulle。
① kan / kunde — 表达能力与请求
用于描述技能、可能性，或礼貌地请求/提供帮助。
✅ 用法示例
能力Jag kan köra bil.
我会开车。
请求Kan du hjälpa mig?
你能帮我吗？
过去Jag kunde inte sova.
我（那时）睡不着。
❌ 常见错误
❌ Jag kan att simma.
✅ Jag kan simma.

❌ Jag kan simmar.
✅ Jag kan simma.

（情态动词后接原形，无att，不变位）
② måste — 义务  ③ ska — 计划/将来
måste表示必须（强义务）；ska表示计划、打算或将来的安排。
Du måste visa legitimation. → 你必须出示证件。
Jag måste inte gå dit. → 我不必去那里。（≠ 不能！）
Vi ska flytta till Malmö nästa år. → 我们明年打算搬去马尔默。
Ska du arbeta imorgon? → 你明天要工作吗？
⚠️ 高频混淆点：måste inte vs. får inte
· måste inte = 不必，没有义务  → Du måste inte komma.（你不必来。）
· får inte = 不允许，禁止  → Du får inte röka här.（你不能在这里抽烟。）
④ vill — 意愿  ⑤ får — 许可  ⑥ bör — 建议  ⑦ behöver — 需要
覆盖日常交流中的意愿、许可、建议和必要性表达。
Jag vill lära mig svenska. → 我想学瑞典语。
Får jag öppna fönstret? → 我可以开窗吗？（请求许可）
Du bör sova mer. → 你应该多睡觉。（温和建议）
Jag behöver inte arbeta på lördag. → 我周六不需要工作。
💬 许可的语气强弱：
· får = 规则/官方层面的允许：Får man parkera här?（这里可以停车吗？）
· kan = 能力/可能性，也常用于礼貌请求：Kan jag sitta här?（我可以坐这里吗？）
· borde（böra的过去时）= 更礼貌的建议：Du borde vila lite.（你应该休息一下。）
✏️ 否定句与疑问句结构
情态动词在否定和疑问语境下的位置规则。
否定句：情态动词 + inte + 动词原形
Jag kan inte simma. → 我不会游泳。
Hon vill inte äta. → 她不想吃。

疑问句：情态动词倒置到主语前（V2规则）
Kan du svenska? → 你会瑞典语吗？
Måste vi gå nu? → 我们现在必须走吗？

💬 对话一：周末安排
蓝色=情态动词，深蓝=动词原形。点击单词可听发音。
🧑
Kan du komma på festen på fredag?
你周五能来派对吗？
👩
Tyvärr kan jag inte. Jag måste jobba sent den kvällen.
很遗憾，我不能来。那天晚上我必须加班。
🧑
Vad synd! Ska du jobba hela helgen?
太可惜了！你整个周末都要工作吗？
👩
Nej, bara på fredag. Men jag vill gärna träffa er snart!
不，只有周五。但我很想尽快和大家聚聚！
🧑
Vi kan träffas på lördag istället. Du behöver inte boka, kom bara!
我们可以改到周六。你不需要预约，直接来就好！
👩
Perfekt! Jag ska definitivt komma då.
太好了！我那时候肯定来。
📝 重点：kan inte（不能）· måste jobba（必须工作）· vill gärna（很想/乐意）· behöver inte boka（不需要预约）
💬 对话二：工作请假
注意许可（får）、建议（borde）和义务（måste）在职场中的实际用法。
👩‍💼
Får jag ta ledigt nästa tisdag?
我下周二可以请假吗？
👨‍💼
Vad ska du göra?
你要去做什么？
👩‍💼
Jag behöver gå till läkaren. Det är ganska viktigt.
我需要去看医生，挺重要的。
👨‍💼
Det borde du absolut kunna göra. Hur länge behöver du vara borta?
你当然应该能去。你需要离开多久？
👩‍💼
Bara ett par timmar. Kan jag börja lite senare på morgonen?
只需要几个小时。我可以早上晚点来吗？
👨‍💼
Ja, det går bra. Du måste bara meddela mig i förväg.
可以，没问题。你只需要提前通知我一下。
📝 重点：Får jag...?（我可以...吗）· behöver gå（需要去）· borde kunna（应该能）· måste meddela（必须通知）
